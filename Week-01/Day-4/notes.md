Concepts Learned

 for Loop
Used when we know how many times we want to repeat a task.

Example:

```python
for i in range(5):
    print(i)
```

### while Loop
Used when we want to repeat a task until a condition becomes False.

Example:

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

 break
Stops the loop immediately.

 continue
Skips the current iteration and moves to the next one.

## Key Takeaways

- `for` loops are useful for fixed repetitions.
- `while` loops continue until a condition changes.
- `break` exits a loop early.
- `continue` skips the current iteration.
- Loops help reduce repetitive code and are essential for solving many programming problems.
