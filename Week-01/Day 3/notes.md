Password Checker

## Objective

The objective of this program is to understand how conditional statements can
be used to verify user input and perform decision-making.

---

## Concepts Learned

- Variables
- User Input
- if Statement
- else Statement
- Comparison Operator (==)
- Decision Making

---

## Program Flow

1. Store the correct password.
2. Ask the user to enter a password.
3. Compare the entered password with the stored password.
4. If both passwords match, display **"Login Successful"**.
5. Otherwise, display **"Incorrect Password"**.

---

## Real-World Applications

- ATM Login Systems
- Mobile App Authentication
- Website Login Pages
- Online Banking Systems
- School Management Systems
- Employee Login Portals

---

## Key Takeaways

- Conditional statements allow programs to make decisions.
- The `==` operator compares two values.
- User authentication is one of the most common uses of `if` and `else`.
