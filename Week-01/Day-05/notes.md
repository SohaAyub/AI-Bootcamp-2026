 Key Concepts

## What is a Function?

A function is a reusable block of code that performs a specific task.

Example:

```python
def greet():
    print("Hello World")
```

---

## Calling a Function

```python
greet()
```

---

## Function with Parameters

```python
def add(a, b):
    return a + b
```

---

## Return Statement

The `return` keyword sends a value back to the place where the function was called.

Example:

```python
def square(number):
    return number * number
```

---

## Built-in Functions

Examples:

- print()
- input()
- len()
- max()
- min()
- type()

---

## User-defined Functions

Functions created by the programmer.

Example:

```python
def welcome():
    print("Welcome to AI Bootcamp")
```
