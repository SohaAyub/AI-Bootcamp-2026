# 🧠 Concepts Learned

## 1. Variables

Variables are used to store information in memory.

Example:

```python
name = "Soha"
age = 22
```

---

## 2. Data Types

Python supports different types of data.

| Data Type | Example |
|-----------|---------|
| String | `"Hello"` |
| Integer | `25` |
| Float | `3.14` |
| Boolean | `True` |

---

## 3. Input

The `input()` function is used to take information from the user.

Example:

```python
name = input("Enter your name: ")
```

---

## 4. Output

The `print()` function displays information on the screen.

Example:

```python
print("Hello World")
```

---

## 5. Comments

Comments make code easier to understand and are ignored by Python.

Example:

```python
# This is a comment
```
